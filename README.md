# Console_RPG
콘솔로 제작된 RPG 게임

## 프로젝트 소개
장비를 장착하고 던전을 돌며 강해지는 콘솔 RPG 게임 입니다.

## 개발 기간
* 23.08.18 ~ 23.08.23

## 멤버 구성
- 강성호 - 게임 시작 화면, 상태보기, 인벤토리(정렬, 장비 장착 관리), 아이템 추가, 콘솔 꾸미기, 인벤토리 크기 맞춤, 상점(구매, 판매),
  장착 개선, 던전(쉬움, 일반, 하드 스테이지), 휴식, 레벨업, 강화

### 개발 환경
- **IDE** : Visual Studio2022

## 주요 기능

#### 시작 화면
    GameStart();
    - 게임 시작 화면
